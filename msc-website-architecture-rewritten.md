**Client:** Pradeep Acharya (BRAINATHON)  
**Website:** magicalstudentsclub.com  
**Primary Goal:** Build trust, showcase programs, and generate WhatsApp enquiries  
**Primary Audience:** Parents of Grade 5+ students and competitive exam aspirants  
**Core Message:** **We don't just tell students to study. We teach them HOW to study effectively.**

---

## 1. Brand Direction

The website should feel **premium, modern, credible, and focused**. Avoid excessive gradients, neon effects, and too many accent colors.

### Color System

```css
--background: #080A0F;
--card: #11141C;
--primary: #E83E8C;
--primary-hover: #F052A0;
--text: #F8F8FA;
--muted-text: #9CA3AF;
--border: #242832;
```

### Color Usage

- **#080A0F** — main page background
- **#11141C** — cards, panels, navbar dropdowns
- **#E83E8C** — primary buttons, key words, icons, active states
- **#F052A0** — button hover and subtle highlights
- **#F8F8FA** — headings and primary text
- **#9CA3AF** — descriptions and secondary text
- **#242832** — borders and dividers

**Rule:** Use pink sparingly. The website should be mostly near-black, white, and muted grey, with pink used only to guide attention.

### Typography

- **Headings:** Space Grotesk
- **Body:** Inter
- Use tabular numerals for statistics such as **90,000+**, **20+**, and **45+**.

# 5. Content Style

Keep website copy:

- Short and easy to scan
- Parent-focused
- Benefit-led
- Credible rather than exaggerated
- Clear enough to understand on mobile

### Copy Rule

Avoid long paragraphs. Use:

- Headline: 5–12 words
- Supporting copy: 1–3 short sentences
- Cards: 15–35 words
- Buttons: 2–5 words

The homepage should build trust in this order:

**Problem → Method → Programs → Process → Coach → Results → Testimonials → FAQ → Contact**

---

# 6. Design Rules

- Maximum content width: **1200–1280px**
- Generous vertical spacing
- Rounded cards: **14–18px**
- Thin borders using **#242832**
- Avoid large gradients
- Use only a subtle pink glow behind important hero imagery
- Use pink for primary actions and important words only
- Keep most cards neutral
- Use consistent icon style
- Keep animations subtle and fast
- Design mobile-first

---

# 7. Performance & Technical Direction

Recommended stack:

- Next.js
- Tailwind CSS
- Optimized WebP/AVIF images
- Lazy-loaded gallery
- Click-to-load video embeds
- Static generation where possible
- GA4 and Meta Pixel
- SEO metadata and schema markup

### Performance Target

- Fast mobile loading
- Minimal heavy animation
- Optimized coach and gallery images
- Avoid loading multiple YouTube iframes immediately

## Final Direction

The website should feel like a **premium education and personal-brand platform**, not a neon technology website.

The visual balance should be approximately:

- **80%** near-black, white, and neutral surfaces
- **20%** deep pink accents

The primary conversion action throughout the website is **WhatsApp**.
